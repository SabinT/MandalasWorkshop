Drawing Mandalas with Code

Choose the setup that works best for you

🌐
Online — p5.js Editor
  1. Go to editor.p5js.org. Sign in, or create a free account if you don't have one.
  2. Open the starter sketch: editor.p5js.org / SabinT / MandalasWorkshop .
  3. Duplicate it to your account: File → Duplicate
  4. Rename your copy: click the ✏️ pencil icon next to the sketch title, type a new name, then press Enter.
  5. Save: File → Save  or  Ctrl S / ⌘ S
  6. Edit motifCustom() in sketch.js to design your mandala, then click ▶ Run to preview.
💻
Local — VS Code
  1. Install VS Code and Git if you haven't already.
  2. Clone the workshop repository: git clone https://github.com/SabinT/MandalasWorkshop.git
  3. Open the folder in VS Code: File → Open Folder… → select MandalasWorkshop
  4. Install the Live Server extension (ritwickdey.LiveServer) from the Extensions panel, or accept the prompt when VS Code offers it automatically.
  5. Right-click index.html in Explorer and choose Open with Live Server. Your browser will open the sketch.
  6. Edit sketch.js and save — the browser refreshes automatically.

💡 All files are plain JS — no build step, no npm required. The p5.js library is bundled locally in library/.
