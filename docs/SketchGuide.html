Understanding the starter sketch and making it your own

📄
The Starter Sketch

All your work happens in sketch.js. The file is organised into three parts: global settings, the setup() / draw() functions provided by p5.js, and one or more motif functions that you design.

sketch.js — annotated

// ── Global settings ───────────────────────────────────────
const MANDALA_BG  = "#accbbc";  // background colour
const CANVAS_SIZE = 1000;        // canvas width & height in pixels
const MOTIF_SIZE  = 10;          // half-extent of motif space (coords: -10 to 10)
const UI = new WorkshopSketchUI(); // toolbar with debug toggle & export

// ── p5.js lifecycle ───────────────────────────────────────
function setup() {
  createCanvas(CANVAS_SIZE, CANVAS_SIZE);
  setMotifSize(MOTIF_SIZE);  // tell the library the motif-space scale
  angleMode(DEGREES);        // use degrees for all angle values
}

function draw() {
  background(MANDALA_BG);
  translate(width / 2, height / 2); // move origin to canvas centre

  // ── Arrange your motifs here ──────────────────────────
  ring({ shape: yourMotif, n: 12, r1: 100, r2: 130 });

  // ── Debug view (toggled from the toolbar) ─────────────
  if (UI.showMotif) {
    showMotif(yourMotif);  // preview motif in isolation
  }
}

// ── Your motif ────────────────────────────────────────────
function yourMotif() {
  // Design your shape here using mLine, mBezier, mTriangle, etc.
  mLine(-10, 10,  10, 10);
  mLine(-10, 10,   0, -10);
  mLine( 10, 10,   0, -10);
}
💡 Click Show Motif from the toolbar to see your motif drawn flat, with the motif-space grid and coordinate labels. This makes it much easier to position shapes precisely.
📐
Key Concepts

Motif

A motif is the small, repeating element of your mandala — a shape or drawing you define as a JavaScript function. It is designed in a flat coordinate space, then automatically placed around a ring.

Motif Space

Motifs are drawn in a square coordinate system called motif space. Both x and y run from −S to +S, where S is the value you pass to setMotifSize(S) (the starter sketch uses S = 10, so coordinates go from −10 to 10).

Think of motif space as a blank tile — you draw whatever shape you like on it, and the library handles the rest.

Axis Range Meaning inside a ring segment
x −S … +S Angular position. −S is the left edge of the segment, 0 is its centre, +S is the right edge.
y −S … +S Radial position. −S sits at the inner radius (r1), +S sits at the outer radius (r2).

Mandala Space

Mandala space is the canvas itself, with the origin at the centre (after translate(width/2, height/2)). Distances are measured in pixels.

The Transformation

When you call ring(), the library maps your flat motif from motif space into polar coordinates on the canvas:

The full circle is divided into n equal angular slices of 360°/n each. For each slice, every point (x, y) in motif space is converted to an angle (from x) and a radius (from y), then placed on the canvas in the correct position. This is what stretches and curves the flat tile into a ring segment.

💡 Because y maps to radius, shapes near y = +S appear at the outer edge of the ring and shapes near y = −S appear at the inner edge.
✏️
Creating a Motif

A motif is a plain JavaScript function. Inside it, you call the motif drawing functions (mLine, mBezier, mCircle, mTriangle, etc.) to describe your shape in motif space. Do not use regular p5.js drawing functions (line(), ellipse(), …) inside a motif — they will not transform correctly.

Step 1 — define a motif function

function motifLeaf() {
  // Left and right Bézier edges form a leaf shape.
  // Coordinates are in motif space (-10 to 10 when MOTIF_SIZE = 10).
  mBezier(0, -10,  -8, -2,  -6, 8,  0, 10);
  mBezier(0, -10,   8, -2,   6, 8,  0, 10);
}

Step 2 — pass it to ring() in draw()

function draw() {
  background(MANDALA_BG);
  translate(width / 2, height / 2);

  stroke(255);
  noFill();
  ring({ shape: motifLeaf, n: 12, r1: 80, r2: 200 });
}
💡 For a full list of available drawing functions — mLine, mBezier, mCircle, mDot, mEllipse, mArc, mTriangle, mQuad, mShape, mPath, mCurve — see the ✏️ Drawing Functions reference page.
🔄
Arranging Motifs in a Ring

The ring() function repeats a motif evenly around a circular band on the canvas. Call it once per ring from inside draw().

ring({ shape, n, r1, r2 })
Parameter Type Description
shape function Your motif function (no parentheses — pass the reference, not the result).
n number Number of repetitions. The full 360° is divided evenly. Try 6, 8, 12, 16, 24.
r1 number (px) Inner radius in pixels (corresponds to y = −S in motif space).
r2 number (px) Outer radius in pixels (corresponds to y = +S in motif space). Must be greater than r1.

Example — a single ring

ring({ shape: yourMotif, n: 12, r1: 100, r2: 200 });

Example — layering multiple rings

function draw() {
  background(MANDALA_BG);
  translate(width / 2, height / 2);

  stroke(255);
  noFill();

  // Inner ring — 8 repetitions, narrow band
  ring({ shape: motifA, n: 8,  r1: 60,  r2: 120 });

  // Middle ring — 12 repetitions, wider band
  ring({ shape: motifB, n: 12, r1: 120, r2: 220 });

  // Outer ring — 24 repetitions, thin accent band
  ring({ shape: motifC, n: 24, r1: 220, r2: 260 });
}
💡 Set stroke colour, fill, and stroke weight with the standard p5.js calls (stroke(), fill(), strokeWeight()) before each ring() call to style rings independently. Wrap each ring in push() / pop() to keep styles isolated.

For full documentation on ring() and setMotifSize(), see the ✏️ Drawing Functions reference page.
