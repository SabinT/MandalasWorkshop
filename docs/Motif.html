Reference

Drawing Functions

Motif-space drawing commands for use inside motif functions

✏️
Overview

A motif is the repeated element of a mandala — a 2D drawing defined in motif space (a square where both x and y run from −S to +S, where S is set by setMotifSize() — default 1).

The functions below are motif drawing functions. Unlike regular p5.js drawing functions, they ensure that everything you draw in motif space transforms correctly into the radial sections of your mandala when rendered with ring().

  • mLine(x1, y1, x2, y2 [, divisions])
  • mBezier(x1, y1, cx1, cy1, cx2, cy2, x2, y2 [, divisions])
  • mCircle(x, y, r [, divisions])
  • mDot(x, y, r)
  • mEllipse(cx, cy, rx, ry [, divisions])
  • mArc(cx, cy, r, startAngle, endAngle [, divisions])
  • mTriangle(x1, y1, x2, y2, x3, y3 [, divisions])
  • mQuad(x1, y1, x2, y2, x3, y3, x4, y4 [, divisions])
  • mShape(points [, divisions])
  • mPath(points [, divisions])
  • mCurve(points [, divisions])
  • mBox(x, y, w, h [, divisions])
  • ring({ shape, n, r1, r2, offset, gapDegrees })
  • setMotifSize(size)

mLine — Straight Line

Draws a straight line between two points in motif space (stroke only).

mLine(x1, y1, x2, y2 [, divisions])
ParameterTypeDescription
x1, y1numberStart point in motif space.
x2, y2numberEnd point in motif space.
divisionsnumber (optional)Number of line segments used to approximate the shape in ring space. Pass 1 to get a single straight (unwarped) segment instead of a smoothly curved one.

Example

function motifCross() {
  mLine(-1, 0,  1, 0);  // horizontal bar
  mLine( 0, -1, 0, 1);  // vertical bar
}

mBezier — Cubic Bézier Curve

Draws a cubic Bézier curve in motif space (stroke only). Arguments match p5.js's bezier() order: anchor 1, control 1, control 2, anchor 2.

mBezier(x1, y1, cx1, cy1, cx2, cy2, x2, y2 [, divisions])
ParameterTypeDescription
x1, y1numberStart anchor point.
cx1, cy1numberFirst control point (influences the curve near the start).
cx2, cy2numberSecond control point (influences the curve near the end).
x2, y2numberEnd anchor point.
divisionsnumber (optional)Number of line segments used to approximate the curve in ring space. Pass 1 to get a single straight (unwarped) segment instead of a smoothly curved one.

Example — a petal-shaped curve

function motifLeaf() {
  // Left edge of petal
  mBezier(0, -1,  -0.8, -0.2,  -0.6, 0.8,  0, 1);
  // Right edge of petal
  mBezier(0, -1,   0.8, -0.2,   0.6, 0.8,  0, 1);
}

mCircle — Circle

Draws a filled circle in motif space. When mapped into a ring, the circle stretches and distorts with the polar transform. Use mDot instead if you want a circle that stays perfectly round in the final mandala.

mCircle(x, y, r [, divisions])
ParameterTypeDescription
x, ynumberCentre of the circle in motif space.
rnumberRadius in motif-space units.
divisionsnumber (optional)Number of polygon segments used to approximate the circle in ring space. Higher values give a smoother result; lower values produce a coarser polygon.

Example — dot at the tip of each segment

function motifDot() {
  mCircle(0, 0.7, 0.15);  // near the outer edge
}

mDot — True Circle

Draws a perfect circle that stays round in both motif space and the final mandala. Unlike mCircle, only the centre is mapped into ring space, so the circle is never distorted by the polar transform. The radius is scaled by the ring's radial extent.

mDot(x, y, r)
ParameterTypeDescription
x, ynumberCentre of the dot in motif space.
rnumberRadius in motif-space units.

Example — perfectly round dot near the outer edge

function motifDot() {
  mDot(0, 0.7, 0.12);  // always a perfect circle in the mandala
}

mEllipse — Ellipse

Draws a filled ellipse in motif space. Like mCircle but with independent horizontal and vertical radii.

mEllipse(cx, cy, rx, ry [, divisions])
ParameterTypeDescription
cx, cynumberCentre of the ellipse in motif space.
rxnumberHorizontal radius.
rynumberVertical radius.
divisionsnumber (optional)Number of polygon segments used to approximate the ellipse in ring space. Higher values give a smoother result; lower values produce a coarser polygon.

Example

function motifOval() {
  mEllipse(0, 0, 0.3, 0.6);  // tall oval centred in the segment
}

mArc — Circular Arc

Draws an open circular arc (stroke only) in motif space.

mArc(cx, cy, r, startAngle, endAngle [, divisions])
ParameterTypeDescription
cx, cynumberCentre of the arc in motif space.
rnumberRadius in motif-space units.
startAnglenumberStarting angle in the sketch's current angleMode() (0 = right, increasing clockwise).
endAnglenumberEnding angle in the sketch's current angleMode().
divisionsnumber (optional)Number of line segments used to approximate the arc in ring space. Pass 1 to get a single straight (unwarped) segment instead of a smoothly curved one.

Example — a smile-shaped arc (degrees)

function motifSmile() {
  mArc(0, 0, 0.5 * MOTIF_SIZE, 0, 180);  // 0° to 180° in DEGREES mode
}

mTriangle — Triangle

Draws a closed, filled triangle in motif space. Specify the three vertices in order (e.g. clockwise).

mTriangle(x1, y1, x2, y2, x3, y3 [, divisions])
ParameterTypeDescription
x1, y1numberFirst vertex.
x2, y2numberSecond vertex.
x3, y3numberThird vertex.
divisionsnumber (optional)Number of line segments per edge used to approximate the shape in ring space. Pass 1 to get straight (unwarped) edges instead of smoothly curved ones.

Example — upward-pointing triangle

function motifArrow() {
  mTriangle(0, -0.8,  -0.4, 0.4,  0.4, 0.4);
}

mShape — Closed Polygon

Draws a closed, filled polygon from an array of vec2 points in motif space.

mShape(points [, divisions])
ParameterTypeDescription
pointsArray<{x, y}>Array of vertices in motif space. Use vec2(x, y) to create each point.
divisionsnumber (optional)Number of line segments per edge used to approximate the shape in ring space. Pass 1 to get straight (unwarped) edges instead of smoothly curved ones.

Example — a pentagon

function motifPentagon() {
  const pts = [];
  for (let i = 0; i < 5; i++) {
    const a = (i / 5) * TWO_PI - HALF_PI;
    pts.push(vec2(cos(a) * 0.6, sin(a) * 0.6));
  }
  mShape(pts);
}

mPath — Open Polyline

Draws an open polyline (stroke only) through an array of vec2 points in motif space.

mPath(points [, divisions])
ParameterTypeDescription
pointsArray<{x, y}>Array of vertices in motif space. Use vec2(x, y) to create each point.
divisionsnumber (optional)Number of line segments per edge used to approximate the path in ring space. Pass 1 to get straight (unwarped) edges instead of smoothly curved ones.

Example — a zigzag path

function motifZigzag() {
  mPath([
    vec2(-1,  0),
    vec2(-0.5, -0.5),
    vec2( 0,   0.5),
    vec2( 0.5, -0.5),
    vec2( 1,   0),
  ]);
}

mCurve — Smooth Spline

Draws a smooth Catmull-Rom spline (stroke only) through an array of vec2 control points in motif space. The curve passes through every point.

mCurve(points [, divisions])
ParameterTypeDescription
pointsArray<{x, y}>Control points in motif space. Use vec2(x, y) to create each point. At least 2 points required.
divisionsnumber (optional)Number of line segments per span used to approximate the spline in ring space. Pass 1 to get straight (unwarped) segments instead of smoothly curved ones.

Example — a flowing wave

function motifWave() {
  mCurve([
    vec2(-1,  0),
    vec2(-0.5,  0.5),
    vec2( 0,  -0.5),
    vec2( 0.5,  0.5),
    vec2( 1,   0),
  ]);
}

mBox — Rectangle

Draws an axis-aligned rectangle in motif space. Fill is respected because the shape is closed.

mBox(x, y, w, h [, divisions])
ParameterTypeDescription
xnumberx-coordinate of the top-left corner in motif space.
ynumbery-coordinate of the top-left corner in motif space.
wnumberWidth of the rectangle in motif-space units.
hnumberHeight of the rectangle in motif-space units.
divisionsnumber (optional)Number of line segments per edge used to approximate the rectangle in ring space. Pass 1 to get straight (unwarped) edges instead of smoothly curved ones.

Example — a centred rectangle

function motifBrick() {
  // A rectangle centred at (0, 0) with width 1.2 and height 0.6
  mBox(-0.6, -0.3, 1.2, 0.6);
}

ring — Radial Repetition

Repeats a motif function evenly around a circular ring on the canvas. Call this from inside draw() after translate(width/2, height/2).

ring({ shape, n, r1, r2, offset, gapDegrees })
ParameterTypeDescription
shape function A motif function that calls motif drawing commands (mLine, mBezier, mCircle, etc.). Pass the function reference — no parentheses.
n number Number of repetitions. The full 360° is divided evenly into n segments. Typical values: 6, 8, 12, 16, 24.
r1 number (px) Inner radius — distance from the canvas centre to the inner edge of the ring, in pixels. Corresponds to y = +S in motif space.
r2 number (px) Outer radius — distance from the canvas centre to the outer edge of the ring, in pixels. Corresponds to y = -S in motif space. Must be greater than r1.
offset number (optional, default 0) Segment offset. Rotates the entire ring by a fraction of one segment. 0.5 shifts by a half-segment — useful for interleaving two rings of the same motif. Defaults to 0.
gapDegrees number (optional, default 0) Angular gap between adjacent elements, in degrees. Shrinks each motif's angular extent so there is empty space between neighbours. Maximum useful value is 360 / n. Defaults to 0.

Example — three concentric rings

function draw() {
  background(20);
  translate(width / 2, height / 2);

  stroke(255);
  noFill();

  // Inner ring — 8 petals, narrow band
  ring({ shape: motifPetal, n: 8,  r1: 40,  r2: 100 });

  // Middle ring — 12 leaves, wider band
  ring({ shape: motifLeaf,  n: 12, r1: 100, r2: 190 });

  // Outer ring — 24 dots, thin outer band
  ring({ shape: motifDot,   n: 24, r1: 190, r2: 220 });
}
💡 The canvas origin (0, 0) must be the centre of the ring. Move it with translate(width/2, height/2) in draw().

setMotifSize — Coordinate Scale

Sets the half-extent of the motif-space coordinate system used by all motif drawing functions. Call this once in setup() before drawing.

setMotifSize(size)
ParameterTypeDescription
size number Half-extent of the design space. Coordinates run from −size to +size on both axes. Defaults to 1. The starter sketch uses 10.

Example — working in a −10 to 10 motif space

const MOTIF_SIZE = 10;

function setup() {
  createCanvas(1000, 1000);
  setMotifSize(MOTIF_SIZE);  // motif coords now range -10 to 10
}

function sampleMotif() {
  mLine(0, -8, 0, 8);  // runs from near inner edge to near outer edge
}
