Reference

Drawing Functions

Motif-space drawing commands for use inside motif functions

✏️
Overview

A motif is the repeated element of a mandala — a 2D drawing defined in motif space (a square where both x and y run from −S to +S, where S is set by setDesignSpace() — default 1).

The functions below are motif drawing functions. Unlike regular p5.js drawing functions, they ensure that everything you draw in motif space transforms correctly into the radial sections of your mandala when rendered with ring().

  • mLine(x1, y1, x2, y2)
  • mBezier(x1, y1, cx1, cy1, cx2, cy2, x2, y2)
  • mCircle(x, y, r)
  • mEllipse(cx, cy, rx, ry)
  • mArc(cx, cy, r, startAngle, endAngle)
  • mTriangle(x1, y1, x2, y2, x3, y3)
  • mQuad(x1, y1, x2, y2, x3, y3, x4, y4)
  • mShape(points)
  • mPath(points)
  • mCurve(points)

mLine — Straight Line

Draws a straight line between two points in motif space (stroke only).

mLine(x1, y1, x2, y2)
ParameterTypeDescription
x1, y1numberStart point in motif space.
x2, y2numberEnd point in motif space.

Example

function motifCross() {
  mLine(-1, 0,  1, 0);  // horizontal bar
  mLine( 0, -1, 0, 1);  // vertical bar
}

mBezier — Cubic Bézier Curve

Draws a cubic Bézier curve in motif space (stroke only). Arguments match p5.js's bezier() order: anchor 1, control 1, control 2, anchor 2.

mBezier(x1, y1, cx1, cy1, cx2, cy2, x2, y2)
ParameterTypeDescription
x1, y1numberStart anchor point.
cx1, cy1numberFirst control point (influences the curve near the start).
cx2, cy2numberSecond control point (influences the curve near the end).
x2, y2numberEnd anchor point.

Example — a petal-shaped curve

function motifLeaf() {
  // Left edge of petal
  mBezier(0, -1,  -0.8, -0.2,  -0.6, 0.8,  0, 1);
  // Right edge of petal
  mBezier(0, -1,   0.8, -0.2,   0.6, 0.8,  0, 1);
}

mCircle — Circle

Draws a filled circle in motif space. The circle is internally approximated with many vertices so it deforms correctly when mapped into polar (ring) space.

mCircle(x, y, r)
ParameterTypeDescription
x, ynumberCentre of the circle in motif space.
rnumberRadius in motif-space units.

Example — dot at the tip of each segment

function motifDot() {
  mCircle(0, 0.7, 0.15);  // near the outer edge
}

mEllipse — Ellipse

Draws a filled ellipse in motif space. Like mCircle but with independent horizontal and vertical radii.

mEllipse(cx, cy, rx, ry)
ParameterTypeDescription
cx, cynumberCentre of the ellipse in motif space.
rxnumberHorizontal radius.
rynumberVertical radius.

Example

function motifOval() {
  mEllipse(0, 0, 0.3, 0.6);  // tall oval centred in the segment
}

mArc — Circular Arc

Draws an open circular arc (stroke only) in motif space.

mArc(cx, cy, r, startAngle, endAngle)
ParameterTypeDescription
cx, cynumberCentre of the arc in motif space.
rnumberRadius in motif-space units.
startAnglenumberStarting angle in the sketch's current angleMode() (0 = right, increasing clockwise).
endAnglenumberEnding angle in the sketch's current angleMode().

Example — a smile-shaped arc (degrees)

function motifSmile() {
  mArc(0, 0, 0.5 * DESIGN_SPACE, 0, 180);  // 0° to 180° in DEGREES mode
}

mTriangle — Triangle

Draws a closed, filled triangle in motif space. Specify the three vertices in order (e.g. clockwise).

mTriangle(x1, y1, x2, y2, x3, y3)
ParameterTypeDescription
x1, y1numberFirst vertex.
x2, y2numberSecond vertex.
x3, y3numberThird vertex.

Example — upward-pointing triangle

function motifArrow() {
  mTriangle(0, -0.8,  -0.4, 0.4,  0.4, 0.4);
}

mQuad — Quadrilateral

Draws a closed, filled quadrilateral in motif space. Specify all four vertices in order.

mQuad(x1, y1, x2, y2, x3, y3, x4, y4)
ParameterTypeDescription
x1, y1numberFirst vertex.
x2, y2numberSecond vertex.
x3, y3numberThird vertex.
x4, y4numberFourth vertex.

Example — a diamond shape

function motifDiamond() {
  mQuad(0, -0.9,  0.4, 0,  0, 0.9,  -0.4, 0);
}

mShape — Closed Polygon

Draws a closed, filled polygon from an array of vec2 points in motif space.

mShape(points)
ParameterTypeDescription
pointsArray<{x, y}>Array of vertices in motif space. Use vec2(x, y) to create each point.

Example — a pentagon

function motifPentagon() {
  const pts = [];
  for (let i = 0; i < 5; i++) {
    const a = (i / 5) * TWO_PI - HALF_PI;
    pts.push(vec2(cos(a) * 0.6, sin(a) * 0.6));
  }
  mShape(pts);
}

mPath — Open Polyline

Draws an open polyline (stroke only) through an array of vec2 points in motif space.

mPath(points)
ParameterTypeDescription
pointsArray<{x, y}>Array of vertices in motif space. Use vec2(x, y) to create each point.

Example — a zigzag path

function motifZigzag() {
  mPath([
    vec2(-1,  0),
    vec2(-0.5, -0.5),
    vec2( 0,   0.5),
    vec2( 0.5, -0.5),
    vec2( 1,   0),
  ]);
}

mCurve — Smooth Spline

Draws a smooth Catmull-Rom spline (stroke only) through an array of vec2 control points in motif space. The curve passes through every point.

mCurve(points)
ParameterTypeDescription
pointsArray<{x, y}>Control points in motif space. Use vec2(x, y) to create each point. At least 2 points required.

Example — a flowing wave

function motifWave() {
  mCurve([
    vec2(-1,  0),
    vec2(-0.5,  0.5),
    vec2( 0,  -0.5),
    vec2( 0.5,  0.5),
    vec2( 1,   0),
  ]);
}

Stroke & Fill

You can use standard p5.js stroke(), fill(), noStroke(), noFill(), and strokeWeight() calls before calling ring() to control how the motif is drawn.

Closed shapes (mCircle, mEllipse, mTriangle, mQuad, mShape) respect both stroke and fill. Open shapes (mLine, mBezier, mArc, mPath, mCurve) are stroke only.

Example

function draw() {
  background(20);
  translate(width / 2, height / 2);

  stroke(200, 100, 255);   // purple stroke
  fill(100, 50, 150, 180);  // semi-transparent fill
  strokeWeight(1.5);
  ring({ shape: motifLeaf, n: 12, r1: 80, r2: 160 });
}
💡 Call push() and pop() around each ring() call if you want to isolate stroke/fill settings for different rings.
